# 🏆 OpenStage

> **Gamify Your Habits. Build Better Routines. Stay Consistent.**

OpenStage is a modern habit tracking application that transforms daily routines into an engaging, game-like experience. Instead of simply checking off tasks, users earn XP, level up, complete streaks, unlock achievements, and visualize their progress through an intuitive dashboard.

Designed with a clean UI and responsive experience, Habit Forge helps users stay motivated while building long-term healthy habits.

---

## 🚀 Features

### 📅 Habit Management

* Create, edit, and delete habits
* Set daily, weekly, or custom schedules
* Track completion history
* Organize habits efficiently

### 🔥 Streak Tracking

* Maintain daily streaks
* View longest streaks
* Monitor consistency over time

### ⭐ Gamification

* Earn XP for completing habits
* Level-up system
* Achievement badges
* Progress indicators
* Motivation through rewards

### 📊 Dashboard

* Habit completion statistics
* XP progress
* Current streaks
* Overall performance summary
* Interactive charts and visualizations

### 🎨 Modern User Interface

* Responsive design
* Clean and intuitive layout
* Smooth animations
* Mobile-friendly experience

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/habit-forge.git
```

Navigate into the project

```bash
cd habit-forge
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser

```
http://localhost:5173
```

---

## 🏗 Build for Production

Create a production build

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
